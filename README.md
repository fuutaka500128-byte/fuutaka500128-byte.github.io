# 博多駅 新幹線案内（スクロール表示）

博多駅の新幹線ダイヤをJSONで管理し、  
index.htmlで電光掲示板風に表示します。

---

## ダイヤ反映ルール（重要）

- ダイヤは `timetable` フォルダ内の **JSON** を編集
- `index.html` は **編集不要**
- ファイル名は必ず **日付形式**

- 表示日は **端末の現在日付と自動連動**

---

## JSON必須フォーマット

```json
{
  "date": "2025-12-27",
  "station": "博多",
  "trains": [
    {
      "direction": "up",
      "service": "nozomi",
      "number": "12号",
      "time": "06:03",
      "destination": "東京",
      "status": "normal",
      "delay": 0
    }
  ]
}

---

## なぜこれで十分か

- ❌ 仕様書を書きすぎる → 読まれない  
- ❌ READMEが長すぎる → 更新されない  
- ✅ **「どこを触るか」「何を書けば動くか」だけ** → 実運用向き

---

## 補足（安心材料）

- 後で  
  - 運休  
  - 遅延  
  - 日付切替  
を追加しても **READMEを書き直す必要はない**
